Web Performance Working Group

test_navigation_type_enums 4.3 The PerformanceNavigation interface This test validates that the TYPE_* enumerations of window.performance.navigation exist and their values are correct. Web Performance Working Group
test_navigation_type_reload 4.3 The PerformanceNavigation interface This test validates the value of window.performance.navigation.type with a reloaded navigation. Web Performance Working Group
test_no_previous_document 4.2 The PerformanceTiming interface This test validates the value of window.performance.timing attributes on an initial navigation. Web Performance Working Group

test_timing_server_redirect 4.2 The PerformanceTiming interface This test validates the values of the window.performance.timing.redirect attributes on a same-origin server redirected navigation. Web Performance Working Group
test_timing_xserver_redirect 4.2 The PerformanceTiming interface This test validates the values of the window.performance.timing.redirect attributes on a cross-origin server redirected navigation. Web Performance Working Group
