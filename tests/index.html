  1. Navigation Timing

Navigation Timing

Test Description
test_document_open This test validates window.performance.timing remains constant when a document is replaced using document.open.
test_navigate_within_document This test validates that all of the window.performance.timing attributes remain unchanged after an in document navigation (URL fragment change).
test_navigation_attributes_exist This test validates that all of the window.performance.navigation attributes exist (but does not validate that their values are correct).
test_navigation_redirectCount_none This test validates that the value of the window.performance.navigation.redirectCount attribute, as well as the window.performance.timing.redirectStart and redirectEnd attributes on a non-redirected navigation.
test_navigation_type_backforward This test validates the value of window.performance.navigation.type with a forward and back navigation.
test_navigation_type_enums This test validates that the TYPE_* enumerations of window.performance.navigation exist and their values are correct.
test_navigation_type_reload This test validates the value of window.performance.navigation.type with a reloaded navigation.
test_performance_attributes_exist This test validates that the navigation and timing attributes exist for window.performance (but does not validate that their values are correct).
test_readwrite This test validates that the window.performance object is read/write.
test_timing_attributes_exist This test validates that all of the window.performance.timing attributes exist (but does not validate that their values are correct).
test_timing_client_redirect This test validates the values of the window.navigation.redirectCount and the window.performance.timing.redirectStart/End times on a client side redirect.
test_unique_performance_objects This test validates that each window has a unique window.performance object.
